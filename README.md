# Task Manager Application

A web-based task management application built with React JS.

## Features

- User authentication with JWT token
- Protected routes
- Task board with To Do, In Progress, and Done columns
- Add new tasks with title, description, priority, status and deadline
- Filter tasks by priority
- Delete tasks and change task status
- Task details page
- Persistent data using localStorage
- Logout functionality

## Tech Stack

- React JS (Functional Components)
- React Router DOM v5
- js-cookie
- CSS

## How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
